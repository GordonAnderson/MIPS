
MIPS host interface commands. These commands are grouped into
catagories. This list includes commands for the common MIPS functions
and modules.

General commands
  NOP               Do nothing
  GVER              Report version
  ISPWR             Report power status
  SUPPLIES          Report main supply voltages
  V12               Report 12 volt supply
  V24               Report 24 volt supply
  CUR               Report input current, in amps
  PWR               Report input power in watts
  UPTIME            Report uptime in mins
  GERR              Report the last error code
  GNAME             Report MIPS system name
  SNAME             Set MIPS system name
  SSERWD            Set serial watch dog time in seconds, 0 to negative to disable
  GSERWD            Return serial watch dog time in seconds
  UUID              Reports the microcontrollers unique ID, 128 bits, hex format
  BIMAGE            Set MIPS boot image
  ABOUT             Report about this MIPS system
  SMREV             Set module rev level, board (a or b),add (hex), rev
  RESET             Reset the Due
  STATUS            Reports the last reboot status and time in millisec from boot
  SAVE              Save MIPS configuration data to default.cfg on SD card
  GCHAN             Report number for the selected system
  MUTE              Turns on and off the serial response from the MIPS system
  ECHO              Turns on and off the serial echo mode where the command is echoed to host, TRUE or FALSE
  TRIGOUT           Generates output trigger on rev 2 and higher controllers
                    supports, HIGH,LOW,PULSE
  AUXOUT            Generates output trigger on Aux output, supports, HIGH,LOW,PULSE
  DELAY             Generates a delay in milliseconds. This is used by the macro functions
                    to define delays in voltage ramp up etc.
  GCMDS             Send a list of all commands
  GAENA             Print the UseAnalog flag, true or false
  SAENA             Sets the UseAnalog flag, true or false
  THREADS           List all threads, there IDs, and there last runtimes
  THREAD            List details on the named thread
  STHRDENA          Set thread enable to true or false
  STHRDINT          Set thread run interval in mS, name, interval (1 to 10000)
  RUNNOW            Run the thread defined by name, now
  THRDRESTART       Restart all threads
  SDEVADD           Set device board and address
  RDEV              Read the ADC channel value, AD7998 device
  RDEV2             Read the ADC channel value, AD7994 device
  TBLTSKENA         Enables tasks in table mode, true or false
  LEDOVRD           Override the LED operation is true, always false on startup
  LED               Define the LEDs state you are looking for.
  DSPOFF            Print the UseAnalog flag, true or false
  STRTDLY           Startup delay in seconds
  CHKIMAGE          Reports the image file status
  LOADIMAGE         Loads an image file to the display
  SSERIALNAV        Set flag to TRUE to enable UI navigation from the host interface
  DIR               List all file in the SD card
  CREATE            Create an empty file, fails if file is present
  ADD               Add a string to the filename passed
  MORE              List file to serial port
  READSD            Read line number from file, 0 is first line
  DEL               Delete file on the SD card
  GET               Dump file contents, hex, from SD card file
  PUT               Create file on SD card from host interface
  SAVEMOD           Save module EEPROM to SD file. Filename,Board (A or B),Add (hex)
  LOADMOD           Load module EEPROM from SD file. Filename,Board (A or B),Add (hex)
  SAVEALL           Saves all the modules EEPROM data to the SD card
  LOADALL           Loads all the modules EEPROM data from the SD card
  GETEEPROM         Sends the selected EEPROM data to the host, Board( A or B), Add (Hex)
  PUTEEPROM         Receives data from the host and writes to EEPROM, Board( A or B), Add (Hex)
  SSPND             Suspend all tasks, suspends real time control, TRUE or FALSE
  GSPND             Returns suspend status, TRUE or FALSE
  CPUTEMP           Returns the CPU temp in degrees C, not an accurate reading
  SSER1ENA          Set the Serial1 port enable, TRUE = general use
  GSER1ENA          Get the Serial1 port enable status
  SETADDRESS        Set memory address to read or write, hex
  SETOFFSET         Set memory address offset to read or write, hex
  SETEEPADD         Set memory address to selected EEPROM buffer, Module name, board address
  WRITE             Write to memory, BYTE,WORD,DWORD,INT,FLOAT
  READ              Read from memory, BYTE,WORD,DWORD,INT,FLOAT
  DUMP              Dump 512 bytes of data in hex format. The address is defined
                    by SETADDRESS or SETEEPADD
  FORMAT            Format the EEPROM, args example,A 0x50,RFdrvA R1
  REEPROM           Read the EEPROM, args board, TWI address, EEPROM address, all data base 10
  WEEPROM           Write the EEPROM, args board, TWI address, address, data, all base 10
  SAVEM             Save the defined module to EEPROM
  STBLRETRIG        Set the table retriggerable flag, TRUE=retriggerable.
  GTBLRETRIG        Returns the table retriggerable flag, TRUE=retriggerable.
  SSETECHO          Set the SerialUSB to Serial1 echo, TRUE enables echo.
  GSERECHO          Return the SerialUSB to Serial1 echo, TRUE enables echo.
  POWER             Cycle MIPS system power
  SUBRDSEL          Set to true to enable the use of board select, default
  GUBRDSEL          Returns the state of the UseBRDSEL flag
  USBPWR            Enables or disables USB powering controller. TRUE = enable. Note if false
                    and power is lost the system will reboot to being powered by USB
  DREAD             Read an arduino pin, returns HIGH or LOW
  DWRITE            Write an arduino pin, Set HIGH or LOW or PULSE or SPLUSE
  DSET              Sets an arduino pin, INPUT, PULLUP, OUTPUT
  DEBUG             Debug function, as needed
  STRPLVL           Set the DCbias power supply readback error trip point in percentage of FS
  GTRPLVL           Returns the DCbias power supply readback error trip point in percentage of FS
TWI commands
  TWIRESET          Resets the TWI interface
  TWIERROR          Reports the numner of detected TWI failures
  STWIHDW           If true then the TWI hardware interface is used for ADC read functions
  GTWIHDW           Returns the current status.
  TWICMD            Sends a command to the addressed TWI (Wire) address, TWICMD,brd,add,string.
                    add is decimal, string is \n terminated
  TWI1CMD           Sends a command to the addressed TWI1 (Wire1) address, TWICMD,add,string.
                    add is decimal, string is \n terminated
  TWISCAN           Scans the selected board address for TWI devices and displays a map (Wire interface)
  TWI1SCAN          Scans the selected board address for TWI devices and displays a map (Wire1 interface)
  TWIADDSET         This function will automatically set the TWI addresses for the selected
                    module. RFD,DCB,FAIMS
  TWITALK           Redirect the serial communications through a board and TWI address passed
  TWI1TALK          Redirect the serial communications through a board and TWI address passed
  STWISPEED         Set TWI speed, 0 | 1, speed
  GTWISPEED         Returns TWI speed, 0 | 1
  GTWI              Reads value from TWI device
  STWI              Writes a value to a TWI device
Log commands
  SLOGENA           Enables logging if TRUE, disable if FALSE
  GLOGENA           Returns the log enable status
  LOGREP            Reports the current log entries
Real time clock functions. Note, no battery backup so the clock reset every time the hardware boots.
  STIME             Sets the current time, 00:00:00 format
  GTIME             Returns the current time, 00:00:00 format
  SDATE             Sets the current date, dd/mm/yyyy format
  GDATE             Returns the current date, dd/mm/yyyy format
Level detection module change enable
  ENALDET           Enables the level detection module to report a detected change to the host.
                    This function requires the module address in hex. OFF to disable
  ENAWLDET          Enables the level detection module to report window state to the host.
                    This function requires the module address in hex. OFF to disable
Pulse counter functions
  COUNT             Setup pulse counter, (Q-X or NA) and level, POS, NEG, BOTH, NA
                    NA disables
  GCNT              Returns the current pulse count
  CLRCNT            Clear the current count
  SCNTTRG           Set the counter threshold count
  GCNTTRG           Return the counter threshold count
  TRIGCNT           If TRUE generate 5uS trigger on Trig out at threshold count
  TRIGRST           If TRUE reset counter out at threshold count
Clock generation functions
  GWIDTH            Report the pulse width in microseconds
  SWIDTH            Set the pulse width in microseconds
  GFREQ             Report the pulse frequency in Hz
  SFREQ             Set the pulse frequency in Hz
  BURST             Generates a frequencyy burst on trig out line
  GMAXFTRIG         Report the maximum frequency for trig out, 0 = nolimit
  SMAXFTRIG         Sets the maximum frequency for trig out, 0 = nolimit
  GMAXFATRIG        Report the maximum frequency for aux trig out, 0 = nolimit
  SMAXFATRIG        Sets the maximum frequency for aux trig out, 0 = nolimit
Delay trigger functions. This supports delayed trigger and retriggering of supported modules
  SDTRIGINP         Set delay trigger input (Q-X) and level, POS or NEG
  SDTRIGDLY         Set trigger delay in uS
  GDTRIGDLY         Returns trigger delay in uS
  SDTRIGPRD         Set trigger delay repeat period in uS
  GDTRIGPRD         Returns trigger delay repeat period in uS
  SDTRIGRPT         Defines the number of trigger repeats, 0 = forever
  GDTRIGRPT         Returns the number of trigger repeats, 0 = forever
  SDTRIGMOD         Defines the delay trigger module, ARB, ADC, SWEEP, AUXTRIG
  SDTRIGENA         TRUE enables the trigger FALSE disables
  GDTRIGENA         Returns the trigger delay enable status
ADC functions
  ADC               Read and report ADC channel. Valid range 0 through 3
  ADCAVG            Report the average ADC value read by the change detection function. This
                    uses the SADCSAMPS value to set the number of samples in the average
  ADCGAIN           Set ADC channel gain. Channel range 0 to 3, gains are 1,2, or 4
  ADCCHG            Enable ADC change detection. Channel range 0 to 3, threshold 0 to 100
  ADCCPRM           Set ADC change filter parameters
  RADCCHG           This function will report ADC value changes to the host, the ADC value is multiplied by gain
  ADCINIT           ADC vector recording setup
  ADCTRIG           ADC vector recording software trigger
  ADCABORT          ADC system abort
  SADCCHAN          Set ADC channel number
  GADCCHAN          Get ADC channel number
  SADCSAMPS         Set ADC number of samples
  GADCSAMPS         Get ADC number of samples
  SADCVECTS         Set ADC number of vectors
  GADCVECTS         Get ADC number of vectors
  SADCRATE          Set ADC sample rate in Hz
  GADCRATE          Get ADC sample rate in Hz
                    ADC record vector to ram buffer commands, implemented Aug 2, 2025
  ADCRBINIT         ADC record vector initilization
  ADCRBTRIG         ADC record vector trigger
  ADCRBREAD         ADC record vector read, start,num
  ADCRBSUM          Reports the sum of all values in the vector recorded
  ADCRBMAX          Reports the maximum value in the vector recorded
  ADCRBVECS         Reports the number of vectors recorded
